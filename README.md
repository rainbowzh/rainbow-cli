## react脚手架，支持ts

#### 如何使用 ： how to use ? 
 
##### install（安装）

 ```
 npm i rainbow-com-cli 
 or
 cnpm i rainbow-com-cli 
 ```

##### create project（创建项目）

```
rainbow-react demo   //demo is dictionary name
```

##### choose alias（相关配置）

>1. is or not a react pro;
>2. is have router
...
